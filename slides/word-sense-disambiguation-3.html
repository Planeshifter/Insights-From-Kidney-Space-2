Similarity Measure

$$ {\scriptstyle \hat c_i = \arg \max_{c_{1} \in C_i} \sum_{w_j \in \mathcal{W}(i)} \max_{c_{2}\in C_j} \operatorname{sim} \left( c_{1}, c_{2} \right) } $$

Jiang-Conrath Similarity Measure:

$$ {\scriptstyle \operatorname{sim}_{JC} \left( c_1, c_2 \right) = - {\color{lorange} IC \left( c_1 \right)} - {\color{lorange} IC \left( c_2 \right)} + 2 \times {\color{lorange} IC(\operatorname{lcs} \left( c_1, c_2 \right) ) } } $$

Information Content of Concept $c$: $$IC \left( c \right) = - \log {\color{lblue}P(c)}$$

Similarity Measure

$$ {\scriptstyle \hat c_i = \arg \max_{c_{1} \in C_i} \sum_{w_j \in \mathcal{W}(i)} \max_{c_{2}\in C_j} \operatorname{sim} \left( c_{1}, c_{2} \right) } $$

Jiang-Conrath Similarity Measure:

$$ {\scriptstyle \operatorname{sim}_{JC} \left( c_1, c_2 \right) = - IC \left( c_1 \right) - IC \left( c_2 \right) + 2 \times IC( {\color{lorange} \operatorname{lcs} \left( c_1, c_2 \right) } ) } $$

Information Content of Concept $c$: $$IC \left( c \right) = - \log P(c)$$

Lowest common subsumer $\operatorname{lcs} \left( c_1, c_2 \right)$: Lowest node in tree that subsumes (is hypernym of) both $c_1$ and $c_2$.

The other part of the definition is the information of the lcs, the lowest common subsumer. The lowest common subsumer of concepts c1 and c2 is defined as the lowest node in tree that is a hypernym of both c1 and c2.
